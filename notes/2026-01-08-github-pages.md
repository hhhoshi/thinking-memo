## GitHub Pages での思考メモ
- GitHub Pages を用いて、日々の思考を整理するための場所を構築したかった。
- noteといったSNSでは、すこし人が多すぎると思ったので、この方法での（半）公開状態を作った

## 運用方法とかの考え
- 日付はあくまでファイル作成の順番の保持のためだから、**修整はいくらでもしてよい**
- 変更履歴はGitHub上に残るので、気にしないで消したり加筆修正していい
- 埋め込みとかもジャンジャンする
- あくまである程度きれいなアウトプットのためであると割り切ることが大事？
